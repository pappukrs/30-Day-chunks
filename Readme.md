# The Core Problem *(samajh lo pehle)*

Speaking mein words na aane ke **3 reasons** hote hain:

1. **Word-to-mouth gap** — Brain mein hai but tongue tak nahi aata fast enough.
2. **Hindi brain** — Pehle Hindi mein socha, phir translate kiya — too slow.
3. **No "chunks"** — Individual words yaad hain, but ready-made phrases nahi hain.

💡 **Solution:** Don't learn words. Learn *phrases* and *sentences* you can reuse.

---

# The Method: "Chunks not Words"

Tech conversations mein same types of sentences baar baar aate hain. Inhe **chunks** bolte hain. Ek baar ye chunks mouth mein fit ho gaye — speaking automatic ho jaati hai.

# 🚀 30-Day Plan

## 🟢 Week 1: Foundation Chunks (Days 1–7)
**Theme:** Explaining your work

| Day | Learn These Chunks | Practice Task |
| :---: | :--- | :--- |
| **1** | "So basically what I'm doing is..." <br/> "The idea here is..." | Apne current task ko 2 min mein explain karo — mirror ke saamne |
| **2** | "I'm working on..." <br/> "Right now my focus is..." | Standup format mein bol — yesterday / today / blocker |
| **3** | "So the issue was..." <br/> "What was happening is..." | Ek recent bug explain karo |
| **4** | "So what I did was..." <br/> "My approach was to..." | Koi bhi solution jo tune implement kiya — explain karo |
| **5** | "Does that make sense?" <br/> "Let me clarify..." | Kisi bhi topic pe 1 min bol, end mein ye phrases use karo |
| **6** | "I'm not 100% sure but I think..." <br/> "I'll need to check that" | Practice saying these without embarrassment — confidence builder |
| **7** | **Review Day** — repeat all 6 days ke chunks ek baar | Record yourself 2 min speaking — listen back |

<br/>

## 🟡 Week 2: Tech-Specific Phrases (Days 8–14)
**Theme:** Code reviews, discussions, questions

| Day | Chunks | Context |
| :---: | :--- | :--- |
| **8** | "So the flow is like this..." <br/> "First it hits the API, then..." | Architecture explain karna |
| **9** | "We have a table called..." <br/> "There's a service that handles..." | Database / backend explain karna |
| **10** | "The problem I'm facing is..." <br/> "I'm stuck at the part where..." | Help maangna confidently |
| **11** | "I was thinking we could..." <br/> "One option is to..." <br/> "Another way would be..." | Suggesting solutions |
| **12** | "That could work, but..." <br/> "My concern is..." <br/> "What if we..." | Pushing back politely |
| **13** | "Just to confirm..." <br/> "So you're saying..." <br/> "Let me repeat that back..." | Active listening phrases |
| **14** | **Review + Record** | Ek fake "tech discussion" — dono sides tum khud bol ke |

<br/>

## 🟠 Week 3: Real Situations (Days 15–21)
**Theme:** Meetings, standups, interviews

| Day | Situation | Key Phrases (Practice) |
| :---: | :--- | :--- |
| **15** | Daily standup | "Yesterday I worked on X. Today I'm going to Y. No blockers." — simple format |
| **16** | Getting a task | "Just to clarify — so you want me to..." <br/> "And the deadline for this is?" |
| **17** | Explaining delay | "So this took longer because..." <br/> "I ran into an issue where..." |
| **18** | Code review | "So the reason I wrote it this way is..." <br/> "Yeah that's a good point, I can change that" |
| **19** | Asking questions in a meeting | "Quick question — when you say X, do you mean...?" |
| **20** | Interview style Q | "So in my last project, we had a situation where..." *(STAR format)* |
| **21** | Record a 3-min mock standup | Pretend camera on hai, team ke saamne bol rahe ho |

<br/>

## 🔴 Week 4: Fluency Push (Days 22–30)
**Theme:** Think in English, not translate

| Day | Exercise | Action Plan |
| :---: | :--- | :--- |
| **22–24** | **"English inner voice"** | Jab koi kaam karo, mentally English mein narrate karo: *"Okay so I'm opening the file... now I need to check the response..."* |
| **25–26** | **Shadowing** | Koi YouTube tech video lo (Fireship, Theo, etc.), pause karo, unhe repeat karo exactly — tone, speed, sab. |
| **27–28** | **Explain like I'm 5** | Koi bhi tech concept (Redis caching, JWT, anything) — simple English mein explain karo as if explaining to a non-tech person. |
| **29** | **Full mock interview** | Record yourself answering: *"Tell me about yourself"* + *"Explain a challenging problem you solved"*. |
| **30** | **Listen back & Compare** | Day 7 recording vs Day 29 recording ko compare karo — growth dikhega! |

---

## 🔑 Daily Habit (5–10 min only)

- **Morning:** Read today's 2-3 chunks out loud *(2 min)*
- **Afternoon:** Use at least 1 chunk in real work/chat
- **Evening:** Speak 1-2 min about your day in English *(voice note to yourself)*

---

## 🧠 The Mindset Shift

❌ *"I need to find the right word"*
✅ ***"I'll say it simply and keep going"***

Jo banda tune suna jo simple words use kar raha tha — woh advanced level hai. Unhone already yeh journey complete kar li hai. **Simple bolna = confident bolna.**
