# 🟢 Day 3: Explaining a Bug/Issue

🎯 **Aaj ka Goal:** Bug ya issue ko explain karna using structured, predictable chunks.

---

## 🗣️ Warm Up
**Pehle in chunks ko warm up karo — 3 baar out loud (zor se) bolo:**

> *"So the issue was..."* <br/>
> *"What was happening is..."* <br/>
> *"So what I found was..."* <br/>
> *"The problem was coming from..."* <br/>
> *"After debugging, I realized..."* <br/>
> *"So to fix this, I..."*

---

## 🛠️ Bug Explain karne ka Structure

- **What was the issue?**     → *"So the issue was..."*
- **What was happening?**     → *"What was happening is..."*
- **Why was it happening?**   → *"The problem was coming from..."*
- **How did you fix it?**     → *"So to fix this, I..."*

---

## 📝 Ab Tera Turn 👇

Mujhe Hindi ya mixed mein bata:

- **Kaunsa bug tune recently fix kiya?**
- **Kya ho raha tha** — user ko kya dikh raha tha ya kya break hua tha?
- **Root cause kya nikla?**
- **Kaise fix kiya?**

💡 *Tera jo aaj ka bug tha — category view wala — woh bhi chal sakta hai! Bata kya exactly issue tha usme?* 👇
